# flask-beginner

Aidemy の「Flask 入門」講座用

# 使い方

環境構築：

```
$ docker-compose up
```

実行：

```
$ docker exec -it <CONTAINER_ID> bash
# python hello.py
-> localhost:5000 へアクセス
```
